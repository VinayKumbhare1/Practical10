# Practical 10 - HTML Iframe

## 📌 Aim
To implement a webpage using HTML iframe and demonstrate embedding of maps and external pages.

## 🎯 Objective
- To understand the use of <iframe> in HTML5
- To embed external websites inside a webpage
- To display Google Maps using iframe
- To integrate multimedia content like videos

## 🛠️ Tools Used
- HTML5
- CSS3
- Visual Studio Code / Notepad
- Web Browser (Chrome)

## 📂 Features of the Project
- Embedded external website (Wikipedia)
- Embedded Google Map (SB Jain College, Nagpur)
- Embedded YouTube video
- Simple and responsive UI design

## ▶️ How to Run
1. Save the file as `index.html`
2. Open the file in any web browser
3. Ensure internet connection is active

## 📍 Output
- Webpage displaying:
  - External website
  - Google Map location
  - YouTube video

## 🧠 Conclusion
This project demonstrates the use of iframe to embed external content such as websites, maps, and videos, making web pages more interactive and user-friendly.
